# 🏥 Hospital Management System

A modern and responsive **Hospital Management System** built using **HTML5, CSS3, JavaScript, PHP, and MySQL**. This project demonstrates Backend API Development, Database Integration, and CRUD Operations.

## ✨ Features

- ➕ Add Patient
- 📋 View Patient Records
- ✏️ Update Patient Details
- ❌ Delete Patient
- 🔍 Search Patient
- 👨‍⚕️ Dynamic Doctor Selection
- 🩸 Blood Group Selection
- 🦠 Disease Selection
- 📅 Admit Date
- 📊 Dashboard Cards

- 🔗 PHP REST API

## ⚙️ How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Open phpMyAdmin.
4. Import `hospital.sql`.
5. Copy the project folder into the `htdocs` folder.
6. Open your browser:

```
http://localhost/HospitalManagementSystem/
```
